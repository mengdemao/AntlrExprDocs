生成法分析树

写法相对固定,携程如下的固定即可

// 输入字符串
ANTLRInputStream input(InputString);
// 词法解析
ExprLexer lexer(&input);
// 分割单词
CommonTokenStream tokens(&lexer);
// 解析语法单元
ExprParser parser(&tokens);
// 生成语法树
ParseTree *tree = parser.prog();
