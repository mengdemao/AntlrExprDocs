◆ visitChildren()

virtual antlrcpp::Any antlr4::tree::AbstractParseTreeVisitor::visitChildren ( ParseTree *  node)
inlineoverridevirtual

The default implementation initializes the aggregate result to defaultResult(). Before visiting each child, it calls shouldVisitNextChild; if the result is

false

no more children are visited and the current aggregate result is returned. After visiting a child, the aggregate result is updated by calling aggregateResult with the previous aggregate result and the result of visiting the child.

The default implementation is not safe for use in visitors that modify the tree structure. Visitors that modify the tree should override this method to behave properly in respect to the specific algorithm in use.
